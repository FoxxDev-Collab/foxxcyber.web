Help & Documentation

Getting Started

This standalone tool helps you assess your organization's compliance with NIST Risk Management Framework (RMF) controls.

How to use this tool:

  1. Start by creating a new assessment from the Dashboard tab.
  2. Enter your organization details in the assessment creation form.
  3. Work through each control family, evaluating the implementation status of each control.
  4. Add detailed notes about your implementation, evidence, and gaps.
  5. Save your progress regularly using the Save button.
  6. Once complete, export your assessment results as JSON for analysis.

Understanding Control Status

Each control can be assigned one of the following statuses:

  • Implemented - The control is fully implemented and effective.
  • Partially Implemented - The control is partially implemented or has some gaps.
  • Planned - Implementation of the control is planned but not yet started.
  • Not Implemented - The control is not implemented.
  • Not Applicable - The control does not apply to your environment.

Saving & Exporting

Your assessments are saved in your browser's local storage. To preserve your work:

  • Export your assessments regularly as JSON files
  • Keep backup copies of your exported files
  • You can import previously exported JSON files back into the tool

Privacy

This is a standalone tool that runs entirely in your browser. Your assessment data never leaves your computer unless you export it and share it. No data is sent to any server.
